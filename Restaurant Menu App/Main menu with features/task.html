Description

Besides the title we would like to list menu items.

Let's start by adding just one menu item below the title.

But we also would like to visually differentiate the title from menu items. Let's use styling and position to signal the difference.

Objectives
  1. Make the existing Title composable be displayed at the center with text size 48dp;
  2. Add a new composable MenuItem that only holds a Text with text "Fettuccine";
  3. Make the composable MenuItem be displayed at the start with text size 24dp;
  4. Display MenuItem bellow Title;
